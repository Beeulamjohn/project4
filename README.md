# 🕊️ Lovebirds Login Page (HTML + CSS)

This project is a **modern login page UI** built with **HTML and CSS**.  
It includes an illustration section on the left and a styled login form on the right with options for email login and Google sign-in.  

---

## 📸 Preview
Lovebirds.png


## 📂 Project Structure

```

project-folder/
│── index.html        # Main HTML file
│── xyz.css           # CSS stylesheet
│── love.png          # Illustration image
│── sign.png          # Google sign-in button image

```

---

## 🚀 Features

- **Two-Column Layout**  
  - Left: Illustration (`love.png`) with heading text.  
  - Right: Login form with branding and inputs.  

- **Login Form**  
  - Input fields for username/email and password.  
  - Rounded **Sign in** button.  
  - "Forgot password?" link.  
  - Divider with "Or".  

- **Google Sign-In Button** (`sign.png`) for alternative login.  

- **Create Account Section** with underlined text link.  

---

## 🎨 Styling (xyz.css)

- **Background**: muted green (`rgb(74, 83, 74)`) with margins.  
- **Flexbox Layout** (`.container`) for side-by-side design.  
- **Image Section**: positioned text overlay on the illustration.  
- **Login Card** (`.border`): white background, rounded edges, clean look.  
- **Typography**: custom font (`Brush Script MT`) for brand title.  
- **Buttons**: grey rounded sign-in button with hover-ready style.  
- **Google Button**: aligned with padding.  

---

## 🛠️ How to Run

1. Clone or download the repository.  
2. Place the required images (`love.png` and `sign.png`) in the project folder.  
3. Open `index.html` in any modern browser (Chrome, Firefox, Edge, etc.).  

---

## 📌 Notes

- This is a **static UI** — no backend authentication is implemented.  
- Replace `love.png` with your own branding image if desired.  
- Add **media queries** in `xyz.css` to improve responsiveness on mobile devices.  
- You can integrate real **Google Sign-In API** if you want full functionality.
